# Grain by Grain

The spine site for the Civilisation of Sand ladder: one public pathway from the Dunwich (Gumpi) ferry terminal to long-horizon civilisation resilience on Minjerribah / North Stradbroke Island, built rung by rung.

Public site target:

- https://auraofintelligence.github.io/grain-by-grain/

Source repo target:

- https://github.com/auraofintelligence/grain-by-grain

## What it is

- A ladder of capability rungs, each anchored by an existing live chapter site (ferry data lab, co-op trust hub, makerspace, tunnelling, clean energy, and more as the project scales).
- Two lanes at every rung: the Build lane (what can be done now) and the Screen lane (what moves to simulation and cinema when cost, law or cultural authority blocks physical work).
- Five evidence labels on every claim: KNOWN, MODELLED, DISPUTED, SPECULATIVE, FICTIONAL.

## What it is not

This is not an official government project page, an engineering approval, a mining proposal, or a cultural authority statement. Minjerribah is Quandamooka Country; QYAC is the public doorway for land and sea decisions and no endorsement is implied.
